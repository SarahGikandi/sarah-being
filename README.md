# Becoming Ess 🌷

A quiet digital space about growth, healing, learning, and becoming.

## ✨ About The Project

**Becoming Ess** is not a traditional portfolio website.

It is a deeply personal and reflective digital space that combines:

* storytelling
* emotional expression
* poetry
* personal growth
* student life
* and creative self-expression



## 🌐 Website Sections

### 🏠 Home

A calm landing page introducing the theme of the website and the journey behind it.

### 👤 My Story

A personal section about growth, identity, struggles, and becoming a better version of myself.

### 🎓 Academics

A reflection on my academic journey, including:

* Calculus
* Statistics
* Programming
* Discrete Structures
* physics

This section highlights both struggles and lessons learned.

### 💻 Projects

A collection of personal coding projects and experiments.

### 🧠 Thoughts

A reflective blog-style section containing thoughts about:

* growth
* healing
* overthinking
* life experiences
* self-discovery

### 📝 Poetry

A personal poetry space featuring emotional and introspective poems.

### ❤️ Relationships

A private, passcode-protected section exploring:

* love
* attachment
* emotional growth
* boundaries
* self-worth

This section is intentionally protected to symbolize emotional privacy and personal boundaries.

### 📷 Gallery

A collection of memories, moments, and visual storytelling.

### 📬 Contact

A functional contact form connected using Formspree.


## 🔐 Special Features

* Passcode-protected Relationships page
* Functional contact form integration using Formspree
* Responsive modern UI
* Smooth transitions and animations
* Emotionally-driven design and storytelling


## 🎨 Design Philosophy

The website was designed to feel:

* soft
* introspective
* calm
* emotional
* minimalistic
* personal

Color palette includes:

* soft creams
* muted purples
* warm neutrals
* subtle gradients


## 🛠️ Built With

* React
* Vite
* Tailwind CSS
* Formspree


## 🚀 Running The Project Locally

Clone the repository:

```bash
git 
# https://github.com/SarahGikandi/sarah-being
```

Navigate into the project folder:

```bash
cd becoming-ess
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
